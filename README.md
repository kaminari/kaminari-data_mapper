# Kaminari::DataMapper

Kaminari DataMapper adapter.


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'kaminari-data_mapper'
```

And bundle.


## Supported versions

* DataMapper 1.1.0+


## Usage

See [Kaminari README](https://github.com/amatsuda/kaminari/blob/master/README.rdoc).
